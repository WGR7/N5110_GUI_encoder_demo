# N5110_GUI_encoder_demo
Simple GUI with Arduino, Encoder and Nokia LCD

# YouTube video:

# Connections:

## Encoder
|Signal|Pin|
|-|-|
|PinA|2|
|PinB|4|
|Button|3|

## Nokia 5110 LCD
|No|Signal | Pin|
|-|-|-|
| 1|RST | 9 |
| 2|CS/CE | 10 |
| 3|DC| 8 |
| 4| MOSI/DIN | 11|
|5| SCK/CLK |  13|
|6| VCC  | Vcc|
|7| LIGHT |6|
|8| GND |GND|

## DHT11 (for demo)
|No|Signal|Pin|
|-|-|-|
|1|VCC|7|
|2|DATA|2|
|3|NC||
|4|GND|GND|
